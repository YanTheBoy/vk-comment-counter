# 'Coca-cola' comment counter
The script was written for getting number of comment from VK witch include 'Coca-cola'.
Then create columnar diagram based on received data.
# How to install
Python3 should be already installed. Then use pip (or pip3, if there is a conflict with Python2) to install dependencies:
```bash
$ pip install -r requirements.txt
```
Collect data and make diagram
```bash
$ python3 com-finder.py
```
